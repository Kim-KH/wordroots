【짧은 설명】 (80자 이내)

어원으로 배우는 영어 단어장. 접두사·어근·접미사 분석으로 어휘력을 근본부터 키우세요.


──────────────────────────────────────
【전체 설명】
──────────────────────────────────────
단어를 통째로 외우는 시대는 끝났습니다.
어원 분석 단어장은 영어 단어의 뿌리를 파고들어, 하나를 알면 열을 이해할 수 있는 어휘 학습을 도와줍니다.

━━ 이런 분께 추천합니다 ━━
✔ 영어 단어를 외워도 금방 잊어버리는 분
✔ 어원을 통해 어휘력을 체계적으로 늘리고 싶은 분
✔ 영어권 학습자뿐 아니라 한국어·일본어·중국어·프랑스어 사용자
✔ 수능, 공인 영어시험(TOEIC·TOEFL·IELTS) 준비생
✔ 언어 구조에 관심 있는 모든 학습자

━━ 핵심 기능 ━━

▶ 어원 중심 단어 학습
  접두사(Prefix), 어근(Root), 접미사(Suffix) 세 파트로 구성된
  2,400개 이상의 영어 단어를 어원별로 분류하여 제공합니다.
  단어 하나하나가 어떤 어원에서 왔는지 한눈에 파악할 수 있습니다.

▶ 형태소 분석 테이블
  각 단어를 접두사·어근·접미사로 분해하고,
  각 요소의 의미를 5개 언어로 제공합니다.
  예) "analogy" → ana-(위로, 다시) + logos(비율, 이성)

▶ 상세 어원 해설
  단어의 유래와 의미 변천 과정을 쉬운 설명으로 제공합니다.
  단순한 정의가 아니라 "왜 이런 뜻이 되었는가"를 이해할 수 있습니다.

▶ 예문 + 번역
  실용적인 영어 예문과 함께 모국어 번역을 제공하여
  문맥 속에서 단어를 자연스럽게 익힐 수 있습니다.

▶ 5개 언어 지원
  한국어 · 영어 · 일본어 · 중국어 · 프랑스어
  언어 전환 버튼 하나로 즉시 전환. 자신의 모국어로 학습하세요.

▶ 발음 듣기 (TTS)
  단어, 예문, 번역을 터치하면 원어민 발음으로 들을 수 있습니다.
  올바른 발음을 귀로 익히며 학습 효과를 높이세요.

▶ 깔끔한 카드 UI
  단어 카드를 터치하면 상세 정보가 부드럽게 펼쳐지는 아코디언 방식.
  필요한 단어만 열어보며 집중력 있게 학습할 수 있습니다.

━━ 수록 내용 ━━
  • 접두사(Prefixes): 225개 단어 수록
  • 어근(Roots): 1,875개 단어 수록
  • 접미사(Suffixes): 364개 단어 수록
  • 총 2,400개 이상의 영어 단어

━━ 사용 방법 ━━
  1. 학습 카테고리 선택 (접두사 / 어근 / 접미사 / 어원)
  2. 어근 그룹 카드에서 관련 단어 목록 확인
  3. 단어 터치 → 어원·형태소·해설·예문 확인
  4. 발음 버튼으로 청취 학습 병행
  5. 우상단 언어 버튼으로 원하는 언어로 전환

"단어 하나를 외우는 것이 아니라,
단어를 만드는 원리를 이해하는 것" — 그것이 어원 분석 단어장의 목표입니다.

【English Description】 (영어권 사용자용)
──────────────────────────────────────

Stop memorizing words. Start understanding them.

Etymology Analysis Vocabulary teaches English words through their roots —
prefixes, roots, and suffixes that unlock the meaning of thousands of words at once.

★ KEY FEATURES
• 2,400+ English words organized by etymological root
• Morpheme analysis: see how each word is built from its parts
• Detailed etymology notes explaining how meanings evolved
• Example sentences with translations
• 5 languages: Korean, English, Japanese, Chinese, French
• Text-to-speech for words, sentences, and translations
• Clean accordion card UI — expand only what you need

★ GREAT FOR
• TOEIC / TOEFL / IELTS vocabulary prep
• Anyone who wants to learn words, not just memorize them
• Language learners of all levels

One root. Ten words. Unlimited vocabulary.
