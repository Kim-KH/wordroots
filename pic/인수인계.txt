========================================
JSON 데이터 형식
========================================

■ 완성된 형식 (Part_Origin 예시)
{
  "root_info": {
    "meaning": { "ko": "떨어져", "en": "away from", "ja": "離れて", "zh": "离开", "fr": "loin de" }
  },
  "words": [{
    "word": "absent",
    "meaning": { "ko": "없는", "en": "absent", "ja": "欠席", "zh": "缺席", "fr": "absent" },
    "analysis": [{
      "type": "접두사 (Prefix)",
      "morpheme": "ab-",
      "meaning": { "ko": "떨어져", "en": "away", "ja": "離れて", "zh": "离开", "fr": "loin" }
    }],
    "note": { "ko": "...", "en": "...", "ja": "...", "zh": "...", "fr": "..." },
    "example": "Why were you absent yesterday?",
    "translation": { "ko": "...", "ja": "...", "zh": "...", "fr": "..." }
  }]
}

■ 미완성 형식 (Part_Prefixes/Roots/Suffixes 현재 상태)
- meaning: { "ko": "~쪽으로", "en": "toward" }
  → ja/zh/fr 없음. 런타임에서 en으로 fallback됨.

- note: { "ko": "...", "en": "", "ja": "", "zh": "", "fr": "" }
  → 한국어 외 빈 문자열. 한국어 이외 설정에서는 note 박스 자체가 안 보임.


========================================
해야 할 작업 (핵심)
========================================

Part_Prefixes / Part_Roots / Part_Suffixes 전체 번역 필요.

현재 scripts/translate-data.js의 DATA_DIR이 Part_Origin만 가리킴:
  const DATA_DIR = path.resolve(__dirname, '../src/data/Part_Origin');

■ 수정 방향

1. DATA_DIR을 Part_Prefixes, Part_Roots, Part_Suffixes 각각에 적용되도록 변경

2. isConverted() 함수를 강화 — ja/zh/fr가 실제로 있는지 확인:
   function isConverted(word) {
     return (
       word.note && typeof word.note === 'object' && word.note.ja
       && word.translation && typeof word.translation === 'object' && word.translation.ja
       && (!word.analysis?.length || word.analysis[0].meaning?.ja)
     );
   }

3. Claude API 프롬프트에서 아래 필드도 번역하도록 추가:
   - analysis[].meaning (각 형태소 의미)
   - word.meaning (단어 자체 의미)
   - root_info.meaning (어근 전체 의미)

4. 번역 후 반드시 실행:
   node scripts/build-etymology-data.js
   → src/data/etymologyData.json 재빌드

■ API 설정
- 모델: claude-haiku-4-5-20251001
- 환경변수 설정: $env:ANTHROPIC_API_KEY = "sk-ant-..."
- 스크립트 실행: node scripts/translate-data.js

========================================
번역 완료 후 확인 명령어
========================================

node -e "
const fs = require('fs');
const data = JSON.parse(fs.readFileSync('src/data/etymologyData.json'));
const cats = ['Part_Prefixes', 'Part_Roots', 'Part_Suffixes'];
for (const cat of cats) {
  let total = 0, ok = 0;
  for (const [f, e] of Object.entries(data)) {
    if (!f.startsWith(cat)) continue;
    for (const w of (e.words ?? [])) {
      total++;
      if (w.note?.ja && w.analysis?.[0]?.meaning?.ja) ok++;
    }
  }
  console.log(cat, ok + '/' + total);
}
"

목표: 세 카테고리 모두 ok/total = 100%
